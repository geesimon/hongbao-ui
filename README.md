# Hongbao-UI

Hongbao is a web3 ZKP application that anyone can setup a donation campaign while keeping the donorsâ€™ activities completely anonymous. 

## Configuration

Update `./static/config.json` with relevant settings.

To specify what configuration set to use, in browser, type `http://<home page url>?env=<main, test, dev>`

## Local Setup

1. `npm  install`
1. `npm start`

## Build
1. `npm  run build`
1. Copy and public content under folder /public to a web server.
